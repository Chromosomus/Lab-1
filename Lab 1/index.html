Elements used:

Element name Description
h1,h3,p Headers and Paragraphs
title Title of the page
table,tr,td,th Specifies a table, it's rows, data and headers
href Specifies a link
img Specifies an image
CSS elements : body,h1,h2 Specifies the properties, such as color, font family etc., of the corresponding elements
CSS element : table,th,td Adds a border to the table
Class .footer Sets the position of any element of this class to the bottom-left of the page
Class .center Sets of the table of this class to the center of the page
